# 🚀 快速入门指南

## ⚡ 30秒快速体验

### 第一步：打开网页
在浏览器中打开项目目录中的 `index.html` 文件

### 第二步：点击登录
在右上角点击"登录"按钮

### 第三步：输入测试账号
选择一个测试身份登录：

| 身份 | 用户名 | 密码 | 功能 |
|------|--------|------|------|
| 👨‍💼 管理员 | `admin001` | `admin123` | 系统管理、用户管理、数据审核 |
| 🌾 农户 | `farmer001` | `farmer123` | 申报处理、处理点查询 |
| ♻️ 回收商 | `recycler001` | `recycler123` | 发布求购、订单管理、财务 |

### 第四步：探索功能
登录后会自动跳转到对应身份的工作台，点击卡片或菜单项浏览功能

## 💡 实用技巧

### 切换用户身份
1. 点击导航栏右上角的用户名
2. 选择"退出登录"
3. 重新登录其他身份的账号

### 添加新用户
1. 在登录弹窗中点击"注册"标签
2. 填写账号、姓名、身份、密码
3. 完成注册后可用新账号登录

### 测试表单功能
1. 以农户身份登录
2. 在侧边栏点击"发起申报"
3. 填写申报表单并提交

## 🔧 常见问题

**Q: 刷新页面后登录状态会丢失吗？**  
A: 不会。刷新页面后仍保持登录状态，除非关闭浏览器标签页。

**Q: 能修改登录账号吗？**  
A: 可以。打开 `auth.js` 文件，修改 `authSystem.users` 对象中的账号信息。

**Q: 怎么连接真实数据库？**  
A: 修改 `auth.js` 中的 `handleLogin()` 函数，改用 AJAX/fetch 调用后端API。

**Q: 各身份的菜单不一样吗？**  
A: 是的。每个身份都有自己的侧边栏菜单和工作台功能。

## 📝 修改提示

### 要改用户名或密码？
编辑 `auth.js` 文件，找到以下代码：
```javascript
users: {
    'admin001': { password: 'admin123', role: 'admin', name: '系统管理员' },
    'farmer001': { password: 'farmer123', role: 'farmer', name: '李农户' },
    'recycler001': { password: 'recycler123', role: 'recycler', name: '王回收商' }
}
```

修改其中的用户名、密码、角色或姓名即可。

### 要改页面样式？
编辑 `index.html` 文件中的 `<style>` 标签，或各个工作台函数中的 `innerHTML` 样式。

### 要添加新菜单项？
1. 在 `auth.js` 的 `updateSidebar()` 函数中添加菜单 HTML
2. 在 `navigateTo()` 函数中添加对应的页面逻辑
